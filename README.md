# python-expense-tracker-CLI
A simple command-line Expense Tracker built using Python.

## Features

- Add new expenses
- View all expenses
- Calculate total expense
- Simple menu-driven program

## Technologies Used

- Python
- List
- Dictionary
- Loops
- Conditional Statements

## How It Works

1. User selects option from menu
2. User enters expense details
3. Data is stored in a list of dictionaries
4. User can view expenses or total expense

## Example Output

===== MENU =====
1. Add Expense
2. View All Expense
3. View Total Expense
4. Exit
